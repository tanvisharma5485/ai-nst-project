# Neural Style Transfer Project

This project implements Neural Style Transfer using PyTorch and a pretrained VGG19 network. It allows users to combine the content of one image with the style of another image to generate a stylized output image.

## Project Description

Neural Style Transfer is a deep learning technique that blends two images:
- Content image: Provides the structure and objects
- Style image: Provides textures, colors, and artistic patterns

The model optimizes an input image to match both content and style representations using a pretrained VGG19 network.

## Features

- Neural Style Transfer using VGG19
- Support for multiple content and style images
- Web interface using Flask (app.py)
- GPU support using CUDA (if available)
- Pretrained model weights included
